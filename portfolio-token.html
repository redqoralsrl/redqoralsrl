구현 기술

- openzeppelin 활용 -
1. 토큰 발행 21000000개
2. hasRole 권한 부여 코드 구현
3. 토큰 생성(_mint), 소각(_burn) 구현
4. 토큰 락업 구성
- 매달 100만개씩 락업 풀리는 코드 구현
- 락업 주인 설정
- 락업 추가물량 락업걸기
- 락업 강제 해제
5. 토큰 전송
- 일반 유저 전송과 관리자 전용 전송 구현
6. 에어드랍 이벤트 그룹 전송 구현
7. 토큰 이름, 심볼, decimals, 개수 조회 구현
8. 토큰 거래 정지 구현(_beforeTokenTransfer)
